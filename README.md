# DecodeLabs Prompt Engineering Project 2

## Project Title

Chain-of-Thought (CoT) Logic Engine

## Objective

The objective of this project is to design a Chain-of-Thought (CoT) prompt that enables an AI model to solve complex logical problems through structured reasoning. The prompt forces the model to think step-by-step, perform self-correction, and provide accurate answers while reducing hallucinations.

## Tools Used

* Python
* Visual Studio Code (VS Code)
* Prompt Engineering
* Chain-of-Thought Reasoning

## Project Description

This project focuses on creating a logical reasoning engine using prompt engineering techniques. The model is instructed to analyze problems carefully, break them into smaller steps, verify its reasoning, and then generate the final answer. The project demonstrates how structured prompting can improve the accuracy and reliability of AI-generated responses.

## Key Features

* Step-by-step reasoning
* Self-correction mechanism
* Logic trap detection
* Hallucination reduction
* Structured cognitive reasoning

## Sample Logic Problems

1. A farmer has 17 sheep. All but 9 die. How many sheep are left?
2. A bat and a ball cost $1.10. The bat costs $1 more than the ball. How much does the ball cost?
3. If five machines take five minutes to make five widgets, how long would 100 machines take to make 100 widgets?

## Expected Outcome

The AI model successfully applies Chain-of-Thought reasoning to solve logic problems accurately, reviews its reasoning process, and provides a verified final answer.

## Learning Outcomes

* Understanding Chain-of-Thought Prompting
* Improving logical reasoning with AI
* Designing self-correcting prompts
* Reducing hallucinations in AI responses
* Building structured AI workflows

## Conclusion

This project demonstrates the effectiveness of Chain-of-Thought prompting in solving multi-step logical problems. By incorporating reasoning and self-correction, the model generates more reliable and explainable answers, making it a valuable technique in prompt engineering.
